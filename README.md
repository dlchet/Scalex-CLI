A command line interface for Scalex
===================================

![Scalex](http://http://scalex.org) is a cool Hoogle style index for Scaladocs.

Run
---

* Clone this project
* Install SBT 0.11
* Start sbt
* Then...
* run List
* run List[A]=>A
* ...
